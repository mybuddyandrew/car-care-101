# Tire Care and Brake Basics - Instructor's Guide

A comprehensive 2-hour class on tire maintenance and basic brake inspection.

## Course Overview
- **Duration**: 2 hours
- **Target Audience**: Basic car knowledge helpful
- **Prerequisites**: None

## Topics
1. Tire Basics
   - Reading tire sizes
   - Pressure monitoring
   - Rotation patterns

2. Tire Wear Patterns
   - Alignment issues
   - Pressure problems
   - When to replace

3. Basic Brake Inspection
   - Pad inspection
   - Rotor condition
   - Warning signs

4. Emergency Tire Change
   - Jack placement
   - Safe procedure
   - Torque specifications
